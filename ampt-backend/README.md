# Ampt Backend

This directory is the [Ampt](https://www.getampt.com) project that serves the web app as static assets via Ampt's global CDN.

## Structure

```
ampt-backend/
├── package.json   # Ampt project config ("ampt": { "static": ["static"] })
├── static/        # Built web app (git-ignored, generated by build:web)
└── README.md
```

## Building & Deploying

**1. Build the web app into `ampt-backend/static/`:**

```bash
# From the repo root:
npm run build:web
```

This runs `expo export --platform web --output-dir ampt-backend/static --clear`.

**2. Deploy to Ampt:**

```bash
cd ampt-backend
npx ampt@latest
```

Ampt will serve `static/index.html` at `/` and all other assets via CDN.
